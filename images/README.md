# Template Images Folder

Place your button template images here:

- **button1.png** - First button to click
- **button2.png** - Second button to click

## How to Create Templates

1. Use a screenshot tool (e.g., Win+Shift+S on Windows)
2. Capture only the button area
3. Save as PNG format
4. Place in this folder

## Notes

- Images should be clear and match exactly what appears on screen
- Background should be included if it's part of the button
- PNG format recommended for transparency support
- Smaller images will process faster
